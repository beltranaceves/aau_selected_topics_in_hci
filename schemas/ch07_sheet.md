# Interactive Surfaces & Spaces

    INTERACTIVE SURFACES
      - 2D position (+ rotation)
      - Input types: multitouch, pen, tokens, blobs

    INTERACTIVE SPACES
      - 6D pose tracking
      - Input: tracked objects/controllers, hand joints, full-body pose

    BIG ISSUES
      - Shared with mobile: touch & gesture problems
      - Unique: infrastructure, fatigue, reachability

    RESEARCH CONTEXT
      - ISS = hardware-heavy research domain
      - Related fields: Tangible Interaction (TEI), Mixed Reality (VRST/VR)

    EXAMPLES
      - SandScape: tangible terrain simulation
      - Reactable: token-based musical instrument
      - Interactive ads: silhouette-based engagement
      - Virtual Valcamonica: archaeological MR exploration
      - SPLOM Wall: large-scale visualization
      - Anatomy MR education: shared spatial learning
